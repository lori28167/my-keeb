# my-keeb

## What I built

A custom mechanical keyboard built from scratch.

## Why

To learn PCB design, case modeling, and firmware and to end up with a keyboard that's truly mine.

## Repo structure

- `pcb/` — KiCad project files (`.kicad_pcb`, `.kicad_sch`) and Gerber exports
  - `pcb/gerbers/` — fabrication output (`.gbr`, `.drl`)
- `case/` — STEP files exported from Onshape (case and plate)
- `photos/` — progress shots and finished build
